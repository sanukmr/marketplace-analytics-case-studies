# Marketplace Analytics Case Studies

This repository contains real-world-inspired case studies focused on
marketplace decision-making, trust, and growth.

The goal is not complex code, but clear analytical thinking and
business-relevant insights.

## Case Studies

### 1. Trust-First Partner Optimisation
Designing a promotion framework that balances revenue with
customer trust using quality gates and confidence-weighted scoring.

Topics:
- Rating confidence vs averages
- Multi-objective ranking
- Marketplace trust systems

### 2. Search Ranking & Sponsored Slots
Analysing trade-offs between monetisation and relevance in
marketplace search using impressions, clicks, and CPC logic.

Topics:
- Sponsored slots
- Auction-based visibility
- Revenue vs experience trade-offs

### 3. City Priority Model
Prioritising city-level investments by combining revenue importance
with customer experience risk.

Topics:
- Weighted decision models
- Resource allocation
- ROI vs churn risk
